== TEH Brog

Brog is a blogging rails application meant to be torn down as an engine for training.

== Requirements

* Ruby 2

* Rails 4

* Mysql

=== Setup

> cp config/database_sample.yml config/database.yml
> bundle
> rake db:create
> rake db:migrate
> rake db:seed

=== Running test

> RAILS_ENV=test rake db:migrate
> rspec

or

> zeus start
> zeus test spec

or 

> guard start
